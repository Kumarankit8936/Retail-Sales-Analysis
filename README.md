# Global Superstore Sales Analysis

## Project Overview

This project analyzes the Global Superstore dataset using MySQL and Power BI.

## Tools Used

- MySQL
- Power BI
- Excel

## Dataset

- Total Orders: 51,290
- Total Customers: 4,873
- Total Products: 10,292

## Key KPIs

- Total Sales: $12.64M
- Total Profit: $1.47M
- Profit Margin: 11.61%

## Dashboard Features

- Sales Analysis
- Customer Analysis
- Product Analysis
- Regional Analysis
- Interactive Filters

## Key Insights

- Central Region generated highest sales.
- Consumer segment contributed highest revenue.
- Technology category was the top-performing category.
